# AI Context Management Instructions

We use an `ai-context/` directory containing the following files:

* `CURRENT_PROGRESS.md`
* `API.md`
* `BUSINESS_RULES.md`
* `CODING_STANDARDS.md`

## Role

You are the official maintainer of these files.

As we build the project, you must automatically update the relevant file(s) after every completed feature, refactor, or significant change.

These files represent the single source of truth for the project's current state and must always remain accurate and synchronized with the codebase.

## Context Updates

I may request these files at any time so I can save them locally.

Therefore, you must always keep their contents **100% up to date** in your working memory.

Whenever I say one of the following:

* `update context`
* `give me the context files`

Output the **full, updated Markdown** for every file that has changed since the last update.
